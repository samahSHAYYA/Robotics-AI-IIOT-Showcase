# Role Index

| Role | Responsibility | Entry point |
|------|---------------|-------------|
| Orchestrator | Plan, delegate, track, accept/reject | Every new task |
| Developer | Implement code, write tests, self-check | After orchestration |
| Reviewer | Check correctness, conventions, security | After dev handoff |
| QA | Validate acceptance criteria, sign off | After review approval |

Full definitions in the corresponding `.md` file for each role.
Workflow connections in `.agent/workflow.md`.
